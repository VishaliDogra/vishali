MongoDB

MongoDB is a source-available cross-platform document-oriented database program. Classified as a NoSQL database program, MongoDB uses JSON-like documents with optional schemas.

Express.js

Express.js is a back end web application framework for Node.js, released as free and open-source software under the MIT License. It is designed for building web applications and APIs.

Angular

Angular is an open-source front-end Typescript library for building user interfaces or UI components. It is maintained by Google and a specifically for single web page applications.

Node.js

Node.js is an open-source, cross-platform, JavaScript runtime environment that executes JavaScript code outside a web browser.
